# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-17

### Added
- **PaperSheet**: A comprehensive export tool for drawings.
    - Export multiple Sheets/Views to PDF and DWG formats.
    - Smart filtering and selection.
    - Custom naming conventions.
    - Automatic paper size detection.
    - Option to merge PDF outputs.
- **Ribbon**: Dedicated "PaperPlus" tab with "Export" and "About" panels.
- **Feedback**: Built-in feedback form.
- **Core**: Support for Revit versions:
    - Revit 2022
    - Revit 2023
    - Revit 2024
    - Revit 2025
    - Revit 2026
