Paper Plus
Paper Plus is a comprehensive Revit Add-in designed to streamline documentation workflows and boost productivity for Revit users.

A powerful batch export utility for Sheets and Views.

Batch Export: Export multiple Sheets/Views to PDF or DWG formats simultaneously.
Smart Filtering: Easily find and select the views you need.
Custom Naming: Define flexible naming conventions for your exported files.
Auto-Size Detection: Automatically detects the correct paper size for PDF exports.
Merge Support: Option to combine multiple sheets into a single PDF or keep them separate.

Installation
Download the latest release.
Run the installer or copy the add-in files to your Revit Addins folder:
%AppData%\Autodesk\Revit\Addins\[Year]\
Launch Revit and look for the PaperPlus tab in the Ribbon.
